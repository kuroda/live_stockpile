# Live Stockpile - a Phoenix LiveView demo

## Setting up the demo

```
$ git clone git@github.com:kuroda/live_stockpile.git
$ cd live_stockpile
$ pushd cd apps/stockpile_web/assets
$ npm i
$ popd
```

## Starting the server

```
$ mix phx.server
```

You can play with this demo by opening http://localhost:4000 with you browser.
